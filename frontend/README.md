# 📦 Vue 3 + TypeScript + Vite + Tailwindcss

## 📑 介绍

> 话不多说, 配置如下:

1. `vueDevTools`
2. `AutoImport`
3. `Components`
4. `alias`
5. `dashboard template`
6. `tailwindcss`

## 🚀 快速开始

### ⚙️ 安装依赖

```bash
bun install
```

### 🖥️ 启动项目

```bash
bun run dev
```

### 🏗️ 构建项目

```bash
bun run build
```

## 📦 使用

```bash
git clone https://github.com/Fromsko/vue-template.git
cd vue-template
bun i
bun run dev
```

### 🎨 自定义主题

请参阅 `src/assets/styles/tailwind.css` 文件进行主题定制。
